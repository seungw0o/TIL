# DFS (Depth-First Search)

DFS는 깊이 우선 탐색 알고리즘으로, 그래프에서 깊은 부분을 우선적으로 탐색한다.
그래프는 노드와 간선으로 이루어진 자료구조이다.

## 탐색 과정

DFS는 스택 자료구조를 이용하며, 탐색 과정은 아래와 같다.

1. 탐색을 시작할 노드를 스택에 삽입하고 방문 처리한다.
2. 스택의 최상단 노드에 방문하지 않은 인접 노드가 있으면 그 노드를 스택에 넣고 방문 처리한다. 방문하지 않은 인접 노드가 없다면 스택에서 최상단 노드를 꺼낸다.
3. 2번 과정을 더 이상 수행할 수 없을 때까지 반복한다.

인접한 노드 중에서 방문하지 않은 노드가 여러 개라면 번호가 낮은 순서부터 처리한다.
코딩 테스트에서는 번호가 낮은 순서부터 처리하도록 명시하는 경우가 종종 있어서, 보통 번호가 낮은 순서대로 구현하는 편이다.

## 구현 예시

아래 코드는 재귀 호출을 이용해 DFS를 구현한 예시이다.
재귀 함수는 호출 스택을 이용하므로 깊이 우선 탐색을 자연스럽게 구현할 수 있다.

```python
def dfs(graph, v, visited):
    # 현재 노드를 방문 처리
    visited[v] = True
    print(v, end=" ")

    # 현재 노드와 연결된 다른 노드를 재귀적으로 방문
    for i in graph[v]:
        if not visited[i]:
            dfs(graph, i, visited)

# 각 노드가 연결된 정보를 리스트로 표현
graph = [
    [],
    [2, 3, 8],
    [1, 7],
    [1, 4, 5],
    [3, 5],
    [3, 4],
    [7],
    [2, 6, 8],
    [1, 7],
]

# 각 노드의 방문 여부를 저장
visited = [False] * 9

# 1번 노드부터 탐색 시작
dfs(graph, 1, visited)
```
