# 음료수 얼려 먹기

`시간 제한 1초 | 메모리 제한 128MB`

## 문제 설명

N x M 크기의 얼음 틀이 있다. 구멍이 뚫려 있는 부분은 `0`, 칸막이가 있는 부분은 `1`로 표시된다. 구멍이 뚫린 부분끼리 상, 하, 좌, 우로 붙어 있으면 서로 연결되어 있는 것으로 간주한다. 이때 얼음 틀의 모양이 주어졌을 때 생성되는 총 아이스크림의 개수를 구하는 프로그램을 작성하시오. 다음의 4 x 5 얼음 틀 예시에서는 아이스크림이 총 3개 생성된다.

```
00110
00011
11111
00000
```

## 입력 조건

- 첫 번째 줄에 얼음 틀의 세로 길이 `N`과 가로 길이 `M`이 주어진다. `(1 <= N, M <= 1000)`
- 두 번째 줄부터 `N + 1`번째 줄까지 얼음 틀의 형태가 주어진다.
- 이때 구멍이 뚫려 있는 부분은 `0`, 그렇지 않은 부분은 `1`이다.

## 출력 조건

- 한 번에 만들 수 있는 아이스크림의 개수를 출력한다.

## 입력 예시

```
00110
00011
11111
00000
```

## 출력 예시

```
3
```

## 추론

핵심은 `0`끼리 연결된 영역의 개수를 세는 것이다.

- `0`이 모여 있는 한 덩어리를 찾을 때마다 카운트를 1 증가시킨다.
- `DFS`를 사용해 현재 위치가 `0`이면 방문 처리한 뒤, 상하좌우로 재귀적으로 탐색한다.
- 이미 방문한 칸은 다시 세지 않으므로 최종적으로 아이스크림의 개수를 구할 수 있다.

## 답안

```python
n, m = map(int, input().split())
graph = []

for i in range(n):
    graph.append(list(map(int, input())))


def dfs(x, y):
    if x < 0 or x >= n or y < 0 or y >= m:
        return False

    if graph[x][y] == 0:
        graph[x][y] = 1
        dfs(x - 1, y)
        dfs(x + 1, y)
        dfs(x, y - 1)
        dfs(x, y + 1)
        return True

    return False


result = 0

for i in range(n):
    for j in range(m):
        if dfs(i, j):
            result += 1

print(result)
```
