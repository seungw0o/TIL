# Template literals

- 백틱(``)으로 감싸며, $와 중괄호({})를 사용해 플레이스 홀더로 표현식을 넣을 수 있다.
- 예시

  ```jsx
  let name = "Hojun";
  let greeting = "Hello my name is " + name;

  console.log(greeting); // Hello my name is Hojun
  ```

  ```jsx
  let name = "Hojun";
  const greeting = `Hello my name is ${name}`;

  console.log(greeting); // Hello my name is Hojun
  ```
