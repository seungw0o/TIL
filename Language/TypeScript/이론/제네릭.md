# 제네릭

```tsx
function getSize(arr: number[] | string[]): number | string {
  return arr.length;
}

const arr1 = [1, 2, 3];
getSize(arr1);
const arr2 = ["a", "b", "c"];
getSize(arr2);
```

- 위처럼 하나씩 추가할 때마다 타입을 추가해야 하는 번거로움이 생긴다.

```tsx
function getSize<T>(arr: T[]): number {
  return arr.length;
}

const arr1 = [1, 2, 3];
getSize<number>(arr1);
const arr2 = ["a", "b", "c"];
getSize<string>(arr2);
```

- 그래서 제네릭을 사용하면 덜 번거롭다. 사용할 때마다 알맞은 타입을 적어 적용하면 된다.
